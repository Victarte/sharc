Detailed Description

Fluid density law with linear expansion.

AI-Generated Documentation:

Computes fluid density as a function of temperature and concentration using the linear expansion model: \(\rho(T,C) = \rho_0 [1 - \beta_T (T - T_0) - \beta_C (C - C_0)]\).
