Check the values of the variable against a reference file.

The reference file filename is an ASCII file containing N values where N is the number of nodes in the mesh.

For each item the current value of node_values is compared to the reference value and if the relative difference is greater than epsilon this is an error.

Only values greater whose absolute value is greater than min_value are checked.
