Detailed Description

AI-Generated Documentation:
Main module for two-phase flow simulation in porous media.

TwoPhaseFlowSimulationModule implements the coupled system of equations for two-phase (water/oil or water/gas) flow, including accumulation, internal flux, and boundary flux computations. It uses the Newton method for solving the non-linear system and supports various physical laws (capillary pressure, relative permeability, fluid density, etc.).
