Detailed Description

Relative permeability power law (Corey-type).

AI-Generated Documentation:

Computes wetting-phase relative permeability using the Corey power-law model: \(k_{rw}(S_w) \propto \bar{S}_w^\alpha\) where \(\bar{S}_w = (S_w - S_{wi}) / (1 - S_{wi} - S_{gc})\), with irreducible wetting saturation \(S_{wi}\), residual non-wetting saturation \(S_{gc}\), and exponent \(\alpha\).
