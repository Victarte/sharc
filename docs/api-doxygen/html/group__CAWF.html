Detailed Description

Compaction, Elasticity, and FEM mechanics.

Finite Element Method modules for solid mechanics and scalar PDEs. Supports 2D and 3D meshes with multiple element types (Tria3, Tetra4, Quad4, Hexa8) and GPU acceleration via SYCL.

Sub-modules:

  • Elasticity: linear elastic deformation \(\nabla\cdot\sigma = f\) with Dirichlet, traction, and body-force BCs (FemModuleElasticity)
  • Poisson: scalar diffusion \(-\nabla\cdot(k\nabla u) = f\) (FemModulePoisson)
  • FEMUtils: shared linear algebra and assembly utilities (DoF systems, BSR/CSR matrices, Gauss quadrature, GPU helpers)
  • DynamicMeshMng: dynamic mesh modifications for sedimentation (ShArc::IDynamicMeshMng)
  • Compaction: sedimentation and compaction with gravitational overburden (CompactionModule)
