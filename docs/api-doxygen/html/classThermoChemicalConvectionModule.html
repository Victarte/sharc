Detailed Description

AI-Generated Documentation:
Thermo-chemical convection simulation module.

ThermoChemicalConvectionModule solves coupled heat and mass transfer with chemical convection in porous media. It handles the non-linear system of equations for pressure, concentration, and temperature, including accumulation, internal flux, and boundary flux terms.
