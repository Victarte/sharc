Detailed Description

Computes Peaceman well indices for well-cell connections.

AI-Generated Documentation:

Implements the Peaceman (1983) well index formula for 3D anisotropic permeability fields. The well index relates the wellbore pressure to the cell-averaged pressure for inflow/outflow calculations.

Formula: \(WI = \frac{2\pi \sqrt{k_x k_y} \Delta z}{\ln(r_e / r_w) + s}\) where \(r_e\) is the Peaceman equivalent radius, \(r_w\) the wellbore radius, and \(s\) the skin factor.
