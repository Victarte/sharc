Detailed Description

Brooks-Corey capillary pressure law.

AI-Generated Documentation:

Computes capillary pressure \(P_c(S)\) as a function of wetting-phase saturation, using the Brooks-Corey model with entry pressure, residual saturations, and pore-size distribution index. Parameters: entry pressure \(P_e\), reference residual \(S_{r,ref}\), residual saturation \(S_r\), and pore-size index \(\lambda\).
