Detailed Description

Provides methods for applying boundary conditions in 3D FEM problems.

This class includes static methods for applying Neumann boundary conditions to the right-hand side (RHS) of finite element method equations in 3D.

Applies a constant source term to the RHS vector.

This method adds a constant source term qdot to the RHS vector for each node in the mesh. The contribution to each node is weighted by the area of the cell and evenly distributed among the number of nodes of the cell.

Parameters
qdotThe constant source term.
meshThe mesh containing all cells.
node_dofDOF connectivity view.
node_coordThe coordinates of the nodes.
rhs_valuesThe RHS values to update.

Applies Manufactured Dirichlet boundary conditions to RHS and LHS.

Updates the LHS matrix and RHS vector to enforce the Dirichlet.

  • For LHS matrix 𝐀, the diagonal term for the Dirichlet DOF is set to 𝑃.
  • For RHS vector 𝐛, the Dirichlet DOF term is scaled by 𝑃.

Applies Neumann conditions to the right-hand side (RHS) values.

This method updates the RHS values of the finite element method equations based on the provided Neumann boundary condition. The boundary condition can specify a value or its components along the x and y directions.
