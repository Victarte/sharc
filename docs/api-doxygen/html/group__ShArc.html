Detailed Description

Main coupling module for ShArc simulation.

The ShArc module orchestrates the coupling between all simulation components. It manages the simulation lifecycle (build, init, time-step loop, finalize), delegates to specialized sub-modules, and handles post-processing and result validation.

Interaction pattern:

  • ShArcModule::build() loads and configures sub-modules
  • ShArcModule::init() initializes mesh, geometry, and sub-modules
  • The time-step loop calls each sub-module's compute methods
  • ShArcModule::endSimulation() finalizes and exports results
