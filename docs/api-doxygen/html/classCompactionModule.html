Detailed Description

Module for compaction and sedimentation simulation.

AI-Generated Documentation:

CompactionModule models sediment deposition and compaction over time, managing sedimentary layers, gravity-driven compaction, and dynamic top boundary evolution. It handles layer events where new sediment is added, and computes density and pressure evolution.
