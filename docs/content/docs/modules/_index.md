ShArc is organized into several modules:

| Module | Description |
|--------|-------------|
| [ShArc]({{< relref "sharc" >}}) | Main orchestrator — initialization, time stepping, post-processing, validation |
| [TwoPhaseFlowSimulation]({{< relref "two-phase-flow" >}}) | Two-phase (water/gas) flow in porous media with Newton solver |
| [ThermoChemicalConvection]({{< relref "two-phase-flow" >}}) | Thermo-chemical convection variant |
| [ExaDiBench]({{< relref "exadibench" >}}) | Mesh geometry benchmarks for exascale co-design |
| [CAWF]({{< relref "cawf" >}}) | Compaction/Elasticity/FEM — Finite Element solvers |
