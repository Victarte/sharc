## Related work

ShArc is built on the Arcane framework and related to the following research:

### Arcane Framework

The Arcane framework provides the component-based simulation platform underlying ShArc. See the [Arcane documentation](https://arcaneframework.github.io/arcane/) for related publications.

### CAWF / FEM

The CAWF module implements finite element methods for geomechanics. The mathematical formulations follow standard linear elasticity and Poisson theory.

### ExaDiBench

ExaDiBench is part of the **NumeX** / **ExaDi** co-design project for exascale computing.
